# Google Maps Reviews Scrapper

## To use:
- Make sure Python3 is installed on your local machine
- Set the PLACE_NAME variable's value with the name of your target place
- Set the REVIEWS_LIMIT variable's value with the number of reviews
- Set the MAPS_LANGUAGE variable's value to enable auto translation (e.g "id" or "en")
- Set the MORE_BUTTON_TEXT variable's value based on the MAPS_LANGUAGE
